PROJECT EXPO: BuildX Hardware Expo

Team Size: 4

Event Description

The ECE Project Expo is a technical showcase highlighting innovative projects developed by Electronics and Communication Engineering (ECE) students.

The expo focuses on practical solutions to real-world engineering challenges.

It provides a platform for students to demonstrate expertise in:

  • Electronics & Communication systems
  • Signal processing
  • Embedded systems
  • VLSI design
  • RF and microwave engineering
  • Internet of Things (IoT)
  • Emerging technologies

Rules

Registration and Eligibility

  • All teams must complete registration before the specified deadline.
  • Registration may include submission of a project abstract, synopsis, or proposal via email.
  • Participants must be bonafide undergraduate students and carry valid college ID cards
  • Each participant is permitted to present only one project.

Project and Model Requirements

  • Preference will be given to working models or functional prototypes
  • Projects must be original; plagiarism in any form will result in disqualification.
  • Projects should highlight social, industrial, or practical relevance
  • Hardware projects must be supported with a presentation (PPT or equivalent)
  • Software projects must be demonstrated using the team's own system/laptop
